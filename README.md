# Projeto Cliente-Servidor com Sockets (Python)

## 📌 Descrição

Este projeto foi desenvolvido para a disciplina de Infraestrutura de Redes.

O objetivo é demonstrar a comunicação entre um cliente e um servidor utilizando sockets TCP na linguagem Python.

O servidor aguarda conexões na porta 5000 e responde mensagens enviadas pelo cliente.

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Biblioteca socket (nativa do Python)
- VS Code
- Git e GitHub

---

## ⚙️ Como Executar o Projeto

### 1️⃣ Instalar o Python
Verifique se o Python está instalado: python --version

---

### 2️⃣ Executar o Servidor

No terminal, dentro da pasta do projeto: python server.py

O servidor ficará aguardando conexão.

---

### 3️⃣ Executar o Cliente

Abra outro terminal e execute: python client.py

Digite mensagens para enviar ao servidor.

Para sair, digite: sair

---

## 🔄 Funcionamento

1. O servidor inicia e fica aguardando conexões.
2. O cliente se conecta ao servidor.
3. O cliente envia uma mensagem.
4. O servidor recebe e responde confirmando o recebimento.
5. A comunicação continua até o cliente digitar "sair".

---

## 📚 Conceitos Aplicados

- Comunicação Cliente-Servidor
- Protocolo TCP
- Sockets
- Endereço IP local (127.0.0.1)
- Porta de comunicação (5000)
